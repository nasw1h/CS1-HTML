# 2. Image as a link

### Aim

> Design a web page to show image as link (include two different links)

### Code

```html
<!DOCTYPE HTML>
<html>

<head>
    <title>Tourist places </title>
</head>

<body bgcolor="violet">
    <center>
        <h1>Tourist places in Malappuram </h1>
        <a href="kot.html">
            <img src="images/kottakunnu.jpg" width="150px" height="150px"></a>
        <a href="teak.html">
            <img src="images/teak.jpg" width="150px" height="150px"> </a>
    </center>
</body>

</html>
```
