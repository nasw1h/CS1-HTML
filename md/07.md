# 7. Course details using hyperlinks

### Aim

> Create a web page to show details (Fee,Duration etc.,) of courses offered by your college using hyperlink(give links to navigate to next,previous and home pages

### Code

07.html
```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>course fee details-home</title>
</head>

<body>
    <h1>
        <font face="castellar" color="red">SIAS COLLEGE
        </font>
    </h1>
    <font color="blue"><u>
            <h3>Courses</h3>
            </u< /font>
            <ul>
                <li><a href="07bca.html">BCA</a></li>
                <li><a href="07bsc.html">BSC COMPUTER SCIENCE</a></li>
                <li><a href="07bcom.html">BCOM</a></li>
            </ul>
</body>

</html>
```
